Debugger for SDN
================

REST Firewall
-------------

*	Run Controller app: ryu-manager examples/Firewall/rest\_firewall.py

*	Run script: sudo ./run.sh

*	Enable firewall and add rules: examples/Firewall/enable.sh, examples/Firewall/add.sh

*	Ping: h1 ping h2
